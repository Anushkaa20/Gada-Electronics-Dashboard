# Gada-Electronics-Dashboard
1.Project Overview

The Gada Electronics Sales Dashboard is an interactive Power BI report designed to analyze and visualize sales performance across different dimensions such as time, brand, customer type, and state.

This dashboard provides key business insights including revenue trends, profitability, order distribution, and geographical performance to support data-driven decision-making.

2.Objective
-The main objective of this dashboard is to:
-Monitor overall sales performance
-Analyze monthly revenue and profit trends
-Identify top-performing brands
-Understand customer segmentation (New vs Returning)
-Track state-wise revenue distribution
-Evaluate order-level details

3.📈 Key Performance Indicators (KPIs)
The dashboard highlights the following key metrics:
-Total Sales: 256M
-Total Profit: 51M
-Profit Margin: 20%
-Total Orders: 2000
-Average Order Value: 127.98K
These KPIs provide a quick summary of overall business performance.

4.📊 Visualizations Used
1️⃣ Monthly Sales & Profit Analysis
Chart Type: Clustered Column Chart
Fields Used:
Axis → Month
Values → Sum of Total Sales
Values → Sum of Total Profit
This visualization shows month-wise comparison of sales and profit to identify seasonal trends and peak performance months.

2️⃣ Sales by Brand
Chart Type: Pie Chart
Fields Used:
Legend → Brand
Values → Sum of Total Sales
This chart helps identify top revenue-generating brands such as Samsung, Apple, Sony, Lenovo, JBL, and Noise.

3️⃣ Orders by Customer Type
Chart Type: Donut Chart
Fields Used:
Legend → Customer Type (New / Returning)
Values → Count of Order ID
This visualization shows customer distribution and helps understand customer retention performance.

4️⃣ State-wise Revenue Analysis
Chart Type: Map Visualization
Fields Used:
Location → State
Size/Color → Sum of Total Revenue
This map displays geographical revenue contribution across states.

5️⃣ Order Details Table
Fields Included:
 Order ID
 Product Name
 Quantity
 Unit Price
 Total
This table provides transactional-level insights for deeper analysis.

5.🛠 Tools & Technologies Used
-Power BI Desktop
-Data Modeling
-DAX (for calculated measures like Profit Margin & Average Order Value)
-Data Cleaning & Transformation (Power Query)

6.📊 Business Insights Derived
-Clear visibility of peak and low-performing months
-Identification of top-selling brands
-Balanced distribution between new and returning customers
-State-level revenue comparison for regional strategy
-Strong overall profit margin (20%) indicating healthy performance
